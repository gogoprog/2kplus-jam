# SHIP2k

## 2kplus-jam

Entry for the 2kPlus Jam : A month long game jam for making 2 kilobyte games.

## Features

    * Title screen/main menu
    * Game can be restarted
    * Highscore
    * HUD
    * Background stars
    * Player ship and enemies ships
    * Particles/explosions
    * Sounds using ZzFX
    * Life bonuses
    * Weapon upgrades

## Build

Use Haxe Compiler 3.4.7
